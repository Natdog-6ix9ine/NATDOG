## Quick Start

### 创建一个名字为`natdog`的hexo仓库

``` bash
hexo init natdog
```
这个步骤会自动npm install 安装的模块会放在文件夹中的`node_modules`文件夹下

### 调试效果

``` bash
hexo cl
hexo g
hexo s
```
